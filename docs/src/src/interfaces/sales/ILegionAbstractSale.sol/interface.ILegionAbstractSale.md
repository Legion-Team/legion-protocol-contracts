```solidity

    uint256 amount,
    ILegionVestingManager.LegionInvestorVestingConfig calldata investorVestingConfig,
    bytes32[] calldata proof
```

|Name|Type|Description|

|`proof`|`bytes32[]`|The Merkle proof for claim verification.|
